This code snippet is an update to automated messages generated in a time management system.  The previously generated messages were long and didn't provide information in a concise format.  Updated the auto-generated messages in the model to to be concise and provide relevant information, approved or denied, in the title without having to open the message.

Files Included:
* MessagesModel.cs - Code in the model that controlls the message generated based on approval or denial.
* MessagesUserView.png - A view of the messages displayed in the system with concise, informative title
* MessageView.cshtml - The code used to populate the messages on page.
